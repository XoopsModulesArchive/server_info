INSTALL INSTRUCTIONS

Just unpack the file to a folder on your server like /sysinfo

Then modify config.php for your needs.

You can write you own theme by coping an excisting one and modify these. If you mail your own theme to me i will take care that the theme is in the next release

If you have the first version of this script you can safely remove uptime.exe. It isn't needed anymore by this script.

When you want to display the apache status & info you must modify your http.conf and put these functions on.

That all there is...

ONLY for Windows NT/2000/XP servers running apache with php.
